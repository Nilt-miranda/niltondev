# Portfólio React

Este é um site de portfólio para programador, criado com React e Vite.

## Como rodar o projeto

1. Instale as dependências:
   ```sh
   npm install
   ```
2. Execute o projeto:
   ```sh
   npm run dev
   ```

## Estrutura de páginas
- **Home:** Apresentação do desenvolvedor
- **Projetos:** Lista de projetos
- **Contato:** Informações para contato

## Personalização
- Edite os arquivos em `src/pages/` para alterar textos e conteúdo.
- As cores e tema estão definidos em `src/styles/theme.css` e `src/styles/colors.js`.

## Dicas
- Cada página contém instruções de edição no próprio código.
- Para adicionar projetos, edite `src/pages/Projetos.jsx`.
- Para mudar seu nome, edite `src/pages/Home.jsx`.
- Para atualizar habilidades, edite `src/pages/Habilidades.jsx`.
- Para alterar o e-mail, edite `src/pages/Contato.jsx`.
